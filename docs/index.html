Documentation - Gappro Admin Theme Template V1.0

Gappro is a responsive admin template that can be used for any backed applications. It's not just a simple replacement of the user interface skins, but strengthen the uitilies of some basic in order to use conveniently. It's coming with black style and beautiful menus, including plenty of useful items, and all the plugin css file has cover to fit the current style. It's building based the bootstrap 2.3, but compatible with the bootstrap 2.2.
Thank you for purchasing this theme. If you have any questions, please feel free to contact me via contact link.

Less is a dynamic stylesheet language.

LESS extends CSS with dynamic behavior such as variables, mixins, operations and functions. You can edit style.less, or edit style.css if you won't using less. There are 25 less files.

  • vars.less, variables for colors and others
  • functions.less, simple functions
  • style.less, gappro style file, include others less
  • global.less, global style
  • background_colors.less, backgrounds and background colors
  • bme.less, bootstrap makrdown editor
  • breadcrumbs.less, breadcrumb path
  • chat.less, chat ui
  • circle_link.less, quick link container and link style
  • compare_box.less, comapre by categories
  • faq.less, faq style and scripts
  • icon_menu.less, main menu
  • main_wrapper.less, main wapper container
  • merge_box.less, summary box with charts
  • messages_box.less, message container style
  • navbar.less, navigations
  • override.less, override booststrap style
  • plugins.less, some plugins style cover
  • status_box.less, compare by colors
  • todo.less, todo list widget
  • typo.less, typography
  • widget.less, widgets, widget table, widget tabs...
  • workspace.less, widget workspace
  • back_top.less, back to top
  • copyright.less, footer and copyright

Config File

You can disable some features that just to set the key-value in config file.

var CONFIG = {
    // menu avatar or logo, false will be remove
    enable_menu_avatar: true,
    
    // filter input box, false will be remove
    enable_menu_filter: true,
    
    // menu collapse/expand, false will be remove
    enable_menu_collapse: true,

    // enable menuitem drag, false will be disabled, cursor: pointer
    enable_menuitem_drag: true,

    // place widget to workspace, false will be remove data-action='thumb'
    enable_widget_workspace: true, 

    // sub-menus, sub-quick-widget trigger way
    menu_sub_trigger: 'click' // click or mouseenter
};
